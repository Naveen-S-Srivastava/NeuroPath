# NeuroPath

NeuroPath is a web-based application designed for visualizing and analyzing the progression of neurological disorders using patient data. It integrates various data sources to provide insights into disease trajectories, aiding clinicians and researchers in understanding disease progression and making informed decisions.

### Prerequisites

Ensure you have the following installed:

* Node.js (v14 or later)
* npm (Node Package Manager)
* MongoDB (for local development) or a cloud database service

### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Naveen-S-Srivastava/NeuroPath.git
   cd NeuroPath
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Set up environment variables:**

   Create a `.env` file in the root directory and add the following:

   ```
   MONGO_URI=your_database_uri
   PORT=3000
   ```

4. **Run the application:**

   ```bash
   npm start
   ```

   The application will be accessible at `http://localhost:3000`.
